汇编笔记

Oct 23, 2018 2 min read 汇编

汇编

指令集简介

x86指令集的基本特色

  • 向下兼容
  • 变长指令
  • 1-15字节，多数为2-3字节长度
  • 多种寻址方式（可访问不对齐内存地址）

指令集的通用寄存器个数有限 x86-32系统下拥有8个通用寄存器（x86-64扩展到16个）

整数的机器表示

机器字长

一般指计算机进行一次整数运算所能处理的二进制数据位数，通常也包括数据地址长度

32位字长

地址的表示空间是4GB，对很多内存需求量大的应用而言，非常有限

机器字在内存中的组织

机器字中的第一个字节的地址，相邻机器字的地址相差4(32-bit)或者8(64-bit)

无符号整数和带符号整数

c语言中的无符号数和带符号数

默认是带符号数，如果有u作为后缀则是无符号数，如0U 如果无符号数与带符号数混合使用，则带符号数默认转换为无符号数，包括比较运算符

汇编与c语言

汇编语言数据结构类型

接下来看一段代码


int sum(int x,int y)
{
 int t =x+y;
 return t;
}

对应的x86-32汇编

Sum:
    Pushl %ebp
    Movl %esp,%ebp
    Movl 12($ebp),%eax
    Addl 8(%ebp),%eax
    Movl %ebp,%esp
    Popl %ebp
ret

int t=x+y;与addl 8(%ebp) %eax语句意思是相同的

c语言 | 汇编语言 | –| –| 两个整数相加| 两个32位整数相加 s| |后缀表示的是双字运算，无符号/带符号整数加法运算指令是一样的

操作数： x: Register eax y: Memory M[ebp+8] t: Register eax 结果存于eax

数据传送指令(mov)

数据传送(AT&T 语法) movl Source,Dest.

将一个”双字”从Source移到Dest

允许的操作数类型:

立即数：常整数 如： $0x400,$-533，可以被1，2或4个字节来表示 寄存器：8个通用寄存器之一

存储器：4个连续字节，支持多种访存寻址模式

数据传送指令支持的不同操作数类型组合

寻址方式

间接寻址 （R） Mem[Reg[R]]

寄存器R指定内存地址

movl (%ecx),%eax

基址+偏移量 寻址 D® Mem[Reg[R]]

寄存器R指定内存起始地址，常数D给出偏移量 movl 8(%ebp),%edx
