抽象类

有些情况下我们需要将问题更加的抽象化，让他更加的通用。比如Person类，Person类的对象基本上没有什么意义。但是由于多态，我们可以通过这个类，能够操纵一系列的类，这就很舒服了。所以，Person类只是表示了接口，而不是一个具体的类。于是Java便引入了一个抽象的机制。

抽象方法

有些方法并不需要实现，它只需要一个声明，其他的由它的子类来实现，这便是抽象方法。

abstract void f();

上面便是它的语法。在返回值前面加上abstract关键字。

包含抽象方法的类就叫做抽象类，如果一个类包含一个或多个抽象方法，这个类就必须被限定为抽象类。

public abstract class Person{
    .....
}

在Class面前加上abstract关键字，这样就将类限定成为一个抽象类。

抽象类的继承

如果一个类想要继承抽象类，并想创建这个类的对象，那么必须实现基类中的抽象方法。

如果不这么做，那么这个类也必须标记为抽象类，也要加上abstract关键字。

当然，我们也可以创建了一个没有任何抽象方法的抽象类。比如有些类之间创建对象是没有什么意义的，我们可以通过此方法阻止它产生它的对象。

实例代码

shape类:

public abstract class shape {
    public shape() {
    }

    public abstract double getArea();

    public abstract String getType();
}

expcet类：

public abstract class expcet extends shape {
    public expcet() {
    }

    public abstract double getGirth();
}

rectangle类：

public class rectangle extends expcet {
    private double length;
    private double height;

    public rectangle(double length, double height) {
        this.height = height;
        this.length = length;
    }

    public double getGirth() {
        return (this.length + this.height) * 2.0D;
    }

    public double getArea() {
        return this.length * this.height;
    }

    public String getType() {
        return "矩形";
    }
}

square类:

public class square extends rectangle {
    private int side;

    public square(double length) {
        super(length, length);
    }

    public void setside(int side) {
        this.side = side;
    }
}

测试类:

public class test {
    public test() {
    }

    public static void main(String[] args) {
        rectangle rectangl = new rectangle(3.0D, 4.0D);
        System.out.println(rectangl.getArea());
        square squar = new square(2.0D);
        System.out.println(squar.getArea());
        System.out.println(rectangl.getType());
    }
}

抽象类利用多态

shape类：

public abstract class shape {
    private String color;

    public abstract double getGirth();

    public abstract String getType();

    public shape(String color) {
        this.color = color;
    }

    public String getcolor() {
        return this.color;
    }

    public void setColor(String color) {
        this.color = color;
    }
}

circle类:

public class circle extends shape {
    private double radius;

    public circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    public void setRadius(double radius) {
        this.radius = radius;
    }

    public double getGirth() {
        return 6.283185307179586D * this.radius;
    }

    public String getType() {
        return "圆" + this.getcolor();
    }
}

Trangle类：

public class Trangle extends shape {
    private double a;
    private double b;
    private double c;

    public Trangle(String color, double a, double b, double c) {
        super(color);
        this.setSide(a, b, c);
    }

    public double getGirth() {
        return this.a + this.b + this.c;
    }

    public String getType() {
        return "三角形";
    }

    public void setSide(double a, double b, double c) {
        if (a < b + c && b < a + c && c < a + b) {
            this.a = a;
            this.b = b;
            this.c = c;
        } else {
            System.out.println("三角形两边之和须大于第三边");
        }
    }
}

测试类:

public class test {
    public test() {
    }

    public static void main(String[] args) {
        shape s1 = new Trangle("白色", 3.0D, 4.0D, 5.0D);
        shape s2 = new circle("黄色", 3.0D);
        System.out.println(s1.getType());
        System.out.println(s1.getGirth());
        System.out.println(s2.getcolor());
    }
}
